# User Stories - GB & IRE Horse Racing Analytics

## User Story 1: Novice Horse Racing Enthusiast
**As a** novice horse racing enthusiast,  
**I want to** view upcoming races with clear information about each horse,  
**So that** I can understand the race without needing to interpret complex newspaper formats.

**Acceptance Criteria:**
- Races are displayed with course name, time, and race name
- Each horse shows horse number and jockey name
- Information is presented in an easy to read format
- No prior knowledge of racing terminology is required

---

## User Story 2: Casual Bettor
**As a** casual bettor,  
**I want to** see which horse is favored to win based on current odds,  
**So that** I can make more informed decisions about potential bets.

**Acceptance Criteria:**
- Favorite horse is clearly marked with a visual indicator
- Non-runners are excluded from favorite calculation

---

## User Story 3: Race Results Explorer
